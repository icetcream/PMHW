## 手柄基础操作说明

本文档当前按 `PS5 / PlayStation` 手柄记号编写，`Xbox` 对应关系如下：

- `三角` = `Y`
- `圆圈` = `B`
- `叉` = `A`
- `方块` = `X`
- `R1` = `RB`
- `R2` = `RT`
- `L1` = `LB`
- `L2` = `LT`

说明：
本项目当前按 `MHW` 风格近战操作编排，并且以大剑原型为主。
后续如果武器、输入映射或 UI 提示发生变化，请以游戏内实际提示为准。

## 大剑原型推荐理解

如果你当前体验的是大剑原型，可以先按下面的思路理解：

- `三角`：主斩击起手，连续按可进入基础连段
- `长按 三角`：进入蓄力斩
- `圆圈`：横扫、肩撞或其他中途派生
- `R2`：防御，适合在来不及翻滚时保命
- `三角 + 圆圈`：肩撞派生入口
- `L2`：强化射击，可用于很多动作后的进一步压制或衔接

## 大剑基础连招示例

以下名称按 `MHW` 常见叫法编写，方便玩家理解。
当前原型中的动作细节、伤害数值和可取消时机，后续仍可能继续调整。

| 输入 | 常见叫法                | 说明 |
| --- |---------------------| --- |
| `三角` | 蓄力斩起手               | 最基础的起手斩击 |
| `三角 -> 任意移动方向 + 三角` | 蓄力斩 -> 强蓄力斩         | 从第一段继续派生到第二段 |
| `三角 -> 任意移动方向 + 三角 -> 任意移动方向 + 三角` | 蓄力斩 -> 强蓄力斩 -> 真蓄力斩 | 大剑最核心的基础三段派生 |
| `长按 三角 -> 长按 三角 -> 长按 三角` | 蓄力斩 -> 强蓄力斩 -> 真蓄力斩 | 原地连续蓄力，适合打硬直或倒地目标 |
| `圆圈 -> 圆圈 -> 圆圈` | 横扫斩 -> 肩撞 -> 飞升跃入斩  | 偏派生向的连续输入，适合中近距离追击与衔接 |
| `蓄力过程中按 圆圈` | 肩撞                  | 可用于顶招、调整节奏并继续派生 |
| `肩撞 -> 三角` | 肩撞派生强蓄力斩/真蓄力斩       | 常用于跳过前段动作，快速进入高伤阶段 |
| `任意攻击后 -> L2` | 强化射击衔接              | 用于压制、补正站位或继续制造派生机会 |


## 连招与派生说明

本项目的连招逻辑参考 `MHW`，核心规则如下：

1. 大多数招式都不是按一次只出一刀，而是在动作进行到特定时机后允许继续派生。
2. 当角色进入可派生阶段时，再次按下 `三角`、`圆圈` 或组合键，就会进入下一段动作。
3. 你不一定要等到动作完全结束再按键，系统支持短时间的预输入缓存。
4. 如果你提前按下了下一次输入，角色会在允许派生的那一刻自动接出下一招。
5. 不同派生会受当前武器状态、是否命中、是否处于蓄力阶段等条件影响。
